# Wordle v2.1 Fixes & UX Enhancements Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Resolve dictionary missing inflections (including "FUERTES"), fix virtual keyboard dark theme styling, fix custom seed game replay duplication, relocate Enviar button to action bar, and add a surrender flag button with a confirmation modal.

**Architecture:** Expand `packages/wordle/src/raeWords.ts` dataset with full Spanish wordforms and dynamic inflection generation. Update `packages/wordle/src/style.css` for dark theme keyboard colors and action bar layout. Refactor `packages/wordle/src/main.ts` for PRNG seed progression, modal dialogs, and submit/surrender button handlers.

**Tech Stack:** TypeScript, HTML5, Vanilla CSS, Vite.

## Global Constraints

- Preserve all existing monorepo build standards (`npm run build:all`).
- Keep `├ü, ├ē, ├Ź, ├ō, ├Ü` as distinct characters when `tildesMode` is enabled; normalize them when disabled, while always preserving `├æ`.
- Maintain dark/light paper theme variables from `@sourceplay/shared`.
- All user-facing text must be in Spanish.
